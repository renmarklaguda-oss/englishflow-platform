# EnglishFlow - Professional ESL Learning Platform

A comprehensive online English language learning platform with student login, teacher dashboard, admin panel, payment processing, video hosting, and certificate system.

## 🌟 Key Features

### 👨‍🎓 Student Features
- ✅ **User Authentication**: Secure student signup and login system
- ✅ **Personal Dashboard**: Customized learning dashboard with progress tracking
- ✅ **Course Management**: Enroll in courses, track progress, and view course materials
- ✅ **Video Learning**: High-quality video player with lesson progression
- ✅ **Certificates**: Automatic certificate generation upon course completion
- ✅ **Progress Tracking**: Detailed analytics and learning metrics
- ✅ **Payment System**: Secure course enrollment with payment processing
- ✅ **Account Settings**: Customize profile, preferences, and payment methods
- ✅ **Transaction History**: View all course purchases and transactions

## 🚀 Getting Started

### Installation
1. Clone or download this repository
2. Open `index.html` in a modern web browser
3. Navigate to signup.html to create an account
4. Login with your credentials

### Demo Credentials
For testing purposes, you can create an account with any email and password (minimum 8 characters).

### File Structure
```
englishflow-platform/
├── index.html              # Homepage
├── login.html              # Student login
├── signup.html             # Student signup

├── styles.css              # Main styling
```
